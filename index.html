The main game system

This system contains slots, battle, inventory, and save/load.

Slots

This is the base of the game system. In slots, you can press spin to either gain money/gems or lose some. Money and gems are two different currencies, but right below the slots there is a shop.

In that shop you can:

  • Exchange the two currencies
  • Rickroll
  • Buy characters

When buying characters, you have 5 options: Common, Uncommon, Rare, Epic, Super rare. If you click on one and have the required gems for it, you can get a character. How to see those characters will be in the next section.

Inventory

In inventory, you can see all of your characters that you have unlocked. Click on one to equip it and see its stats.

Battle

But how do you use those characters? Well, in battle you can battle a CPU opponent using your selected character. Press next for you or the CPU to take a turn in battle, taking away HP from their opponent. If you make the CPU's health go to 0, you win you will get a reward. If you win or lose, the next button will function as a "new opponent" button for the next click.

Here is what all of the stats mean:

  • HP: How much health it has. More health usually means a better character, because it can battle longer.
  • Range: The range of the damage (HP taken away) to the opponent. Every time you click next, you deal a random number in the range of damage.
  • Attack: The name of the attack it uses.
  • Rarity: Common, Uncommon, Rare, Epic, Super rare, and Legendary. The higher the rank is, the better the character is.

Save/Load

Now that I added automatic save/load, there is really no use for this except for these reasons:

  • Loading older save codes in
  • Device transfer
  • Keeping the code if for whatever reason the automatic save/load fails.

How to device transfer:

  1. On the sending device, write down the save code if you want.
  2. On the receiving device, type in the box the save code.
  3. On the receiving device, press load.

It should work. You can also use older save codes if they are written down.

Advanced features and tips about the game system

These might help you in the game system:

  1. Writing down the save code often. The automatic save/load has not been tested a lot, and if history is cleared, it might not work.
  2. Slots strategy. As soon you get 3 gems, try cashing them in for money, and do that for a while until you get around $800. Then, you should be safe and spam spin until you get enough gems to buy a character.
  3. Progress through ranks/rarities. You should buy a common/uncommon first, battle for a while to get $ and gems, then get a higher rarity one, battle again, etc, until you either are bored or want to get all of the characters.
